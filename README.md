### Folder Description

```
.
├── public
│   └── images                  # 頁面使用的 img
├── src
│   ├── assets/svg              # 使用 icon

│   ├── constants/              # 管理設定
│   ├── jotai                   # Jotai 管理
│   ├── page/                   # 對應的頁面
│   ├── route                   # Route 設定

```
